# 🏆 WorldCup-Intelligence-Skill

世界杯智能分析与内容生成 Skill，为 AI Agent 提供专业足球赛事分析能力。

## 功能亮点

- 🔍 **赛前情报**：伤病、阵型、历史交锋一网打尽
- ⚽ **战术解读**：阵型拆解、对位分析、变阵预判
- 📊 **比赛预测**：多维度加权概率分布
- ⚠️ **爆冷预警**：冷门触发因子识别
- 📝 **赛后复盘**：关键事件时间线 + 转折点分析
- 🎬 **内容生成**：公众号/短视频/社交媒体全覆盖

## 核心链路

```
球队画像 → 赛前情报 → 战术对位 → 关键球员 → 比赛预测 → 爆冷预警 → 赛后复盘 → 内容生成
```

## 快速使用

1. 将本仓库作为 Skill 加载
2. 选择对应的 prompt 模式（分析师/教练/解说员/数据/媒体创作）
3. 提供比赛信息（对阵双方、赛事阶段）
4. 获取结构化分析输出

## 目录说明

| 目录 | 内容 |
|------|------|
| `examples/` | 实际使用示例（赛前分析、战术分析、小组推演、短视频脚本） |
| `templates/` | 标准化输出模板（可直接填充使用） |
| `prompts/` | 角色模式提示词（决定分析风格与输出格式） |

## 约束声明

- 不虚构数据，统计必须标注来源
- 预测以概率表达，不做绝对化断言
- 不提供博彩建议
- 伤病信息以官方通报为准
