# WorldCup-Intelligence-Skill

## 项目简介

这是一个面向 FIFA 世界杯赛事的中文 AI Skill，覆盖赛前情报分析、战术解读、可解释预测、爆冷预警、赛后复盘、小组出线推演和多平台内容生成的完整链路。

本 Skill 设计为可被任意 AI Agent 加载使用的标准化能力模块，提供结构化的分析框架、输出模板和角色提示词，使 Agent 能够以专业足球分析师的水准输出赛事内容。

## 核心亮点

- 🔌 **零 API 依赖**：不要求用户额外配置任何外部 API，开箱即用
- 🌐 **实时信息联网核验**：遇到实时数据（比分、积分榜、伤停）默认联网搜索权威来源，不凭记忆编造
- ⚽ **赛前分析 + 赛后复盘**：覆盖比赛全生命周期
- 🧩 **战术对位矩阵**：逐位置对比双方攻防优劣
- 📊 **可解释预测**：所有预测必须说明依据、置信度和不确定因素
- ⚠️ **爆冷风险识别**：基于历史模式匹配和当前触发因子
- 🎬 **多平台内容生成**：支持短视频脚本、公众号长文、直播解说词、社交媒体文案

## 适用场景

| 用户类型 | 使用方式 |
|----------|----------|
| 普通球迷 | 快速了解比赛看点、获取赛前预测和赛后复盘 |
| 体育内容创作者 | 生成短视频脚本、公众号文章、社媒文案 |
| 赛事分析人员 | 结构化数据分析、战术对位、小组推演 |
| AI Skill / Agent 开发者 | 作为可复用能力模块集成到自有 Agent |
| 教学和案例演示 | 展示 AI Skill 的标准化设计方法 |

## 能力模块

| 模块 | 作用 |
|------|------|
| 赛前情报分析 | 分析比赛背景、双方状态、伤停、首发、胜负手 |
| 球队画像分析 | 分析国家队整体实力、战术风格、阵容深度 |
| 球员影响力分析 | 分析关键球员的技术特点和战术作用 |
| 战术对位矩阵 | 对比两队在门将、中卫、中场、边路、反击、定位球等方面的优劣 |
| 可解释预测 | 给出比赛倾向、依据、不确定因素和爆冷风险 |
| 赛后复盘 | 分析比赛转折点、战术执行、球员表现和胜负原因 |
| 小组出线推演 | 根据积分榜、净胜球和剩余赛程分析晋级条件 |
| 内容生成 | 生成短视频脚本、公众号文章、解说词和社媒文案 |

## 数据来源策略

**本 Skill 默认不要求用户配置外部 API。**

实时信息处理按以下优先级获取与核验：

```text
用户提供的数据
↓
FIFA 官方信息
↓
球队或足协官方公告
↓
主流体育媒体报道（ESPN、BBC Sport、懂球帝、直播吧）
↓
数据平台（FBref、Transfermarkt、WhoScored）
↓
历史赛事资料
↓
模型已有知识
```

对于最新赛程、比分、积分榜、首发、伤停、淘汰赛对阵等信息，**必须联网核验**，不允许凭记忆编造。所有引用数据必须标注来源或注明截止日期。

## 快速使用

将本仓库作为 Skill 加载后，直接用自然语言提出需求即可：

```text
使用 WorldCup-Intelligence-Skill，分析阿根廷 vs 法国的赛前看点。
```

```text
使用 WorldCup-Intelligence-Skill，根据以下积分榜推演小组出线形势。
```

```text
使用 WorldCup-Intelligence-Skill，生成一条世界杯爆冷比赛的短视频脚本。
```

```text
使用 WorldCup-Intelligence-Skill，以教练模式分析西班牙 4-3-3 vs 德国 3-4-2-1 的战术对位。
```

## 输出示例结构

以赛前分析为例，标准输出包含以下模块：

```text
一、核心结论
二、信息来源与时间状态
三、双方实力对比
四、战术对位分析
五、关键球员
六、胜负手
七、预测结论
八、爆冷风险
九、不确定因素
```

具体示例详见 `examples/` 目录。

## 约束规则

* 不虚构实时信息
* 不凭记忆回答最新比分、积分榜、首发、伤停
* 涉及实时信息必须联网核验
* 预测必须说明依据和不确定性
* 不提供赌博、投注金额或下注建议
* 信息冲突时优先采用官方来源
