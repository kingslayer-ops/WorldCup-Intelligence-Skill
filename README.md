## 项目简介

**WorldCup Intelligence Skill** 是一个结构化的 AI 能力模块，专为世界杯赛事分析设计。它为 AI Agent 提供标准化的分析框架、输出模板、角色提示词和数据核验规范，使 Agent 能够以专业足球分析师的水准输出赛事内容。

本项目 **不是博彩工具**，不提供任何投注建议。所有预测仅为分析参考，不是确定结论。

| 特性 | 说明 |

## 数据来源策略

本 Skill 遵循严格的数据核验规范（详见 [`SOURCE_POLICY.md`](./SOURCE_POLICY.md)）：

```
用户提供数据 → FIFA 官方 → 各国足协 → 权威体育媒体 → 数据平台 → 模型已有知识（仅补充）
```

**核心原则**：
- 实时信息（比分、积分榜、首发、伤停）**必须联网核验**
- 若无可靠来源，必须标注"信息不足"或"不确定"
- 所有引用数据必须标注来源和截止时间
- 信息冲突时官方来源优先，不一致必须标注

## 输出示例结构

以赛前分析为例，标准输出包含以下模块：

| 序号 | 模块 | 说明 |
|------|------|------|
| 1 | 核心结论 | 一句话总结分析倾向 |
| 2 | 信息来源与时间状态 | 数据截止时间、来源列表、数据状态 |
| 3 | 比赛背景 | 赛事阶段、历史交锋 |
| 4 | 双方状态 | 近期战绩、士气、体能 |
| 5 | 阵容与伤停 | 预计首发、关键缺阵 |
| 6 | 战术对位 | 阵型匹配、突破口 |
| 7 | 关键球员 | 攻防两端核心人物 |
| 8 | 胜负手 | 决定比赛走向的关键因素 |
| 9 | 预测倾向 | 概率区间 + 依据 + 不确定性 |
| 10 | 爆冷风险 | 冷门触发因子和风险等级 |
| 11 | 不确定因素 | 信息缺失、未确认事项 |

具体示例详见 [`examples/`](./examples/) 目录。

## 约束规则

1. **不虚构实时信息** — 所有统计必须标注来源或注明截止时间
2. **联网核验** — 涉及比分、积分榜、首发、伤停必须联网确认
3. **可解释预测** — 概率使用区间表达，必须说明依据和不确定性
4. **禁止博彩建议** — 不提供任何赌博、投注金额或下注建议
5. **官方优先** — 信息冲突时优先采用官方来源
6. **标注不确定性** — 无法确认的信息必须标注"不确定"或"信息不足"
7. **预测非结论** — 预测结果只能作为分析参考，不是确定事实
