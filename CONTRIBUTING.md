# 贡献指南

欢迎为 WorldCup Intelligence Skill 做出贡献！🎉

---

## 可贡献内容

| 贡献类型 | 说明 |
|----------|------|
| 新模板 | 新的分析输出模板（如球员对比模板、转会影响模板） |
| 新示例 | 新的 mock 格式示例或 real 历史案例 |
| 新角色模式 | 新的提示词角色（如"球迷模式""记者模式"） |
| 数据来源规范 | 补充或修正 SOURCE_POLICY 中的来源信息 |
| 真实历史案例 | 使用真实公开数据的分析案例 |
| 文档改进 | 修正错误、补充说明、优化结构 |
| 翻译 | 英文版本或其他语言 |

## 提交规范

### 示例文件命名

- **虚构示例**：以 `mock_` 开头，如 `mock_赛前分析示例.md`
- **真实案例**：以 `real_` 开头，如 `real_2022决赛_阿根廷vs法国.md`

### 虚构示例要求

1. 文件开头必须包含免责声明：

```markdown
> ⚠️ **注意**：本示例使用虚构或简化数据，仅用于展示输出格式，不代表真实赛事信息。
```

2. 使用匿名球队名（"球队 A/B"）或明确标注"虚构"
3. 概率使用区间表达
4. 不使用看起来像真实未来赛事的确定性说法

### 真实案例要求

1. 必须基于已结束的比赛（历史数据）
2. 标注数据来源（FIFA.com / FBref / ESPN 等）
3. 标注数据状态（✅ 官方确认 / 📰 媒体报道）
4. 文件开头说明"本文件使用真实历史比赛数据"

### 不接受的内容

| 类型 | 原因 |
|------|------|
| 博彩导向内容 | 投注建议、赔率分析、下注策略 |
| 虚构的"真实案例" | 使用未来比赛编造数据 |
| 无来源的数据 | 所有事实性数据必须可查证 |
| 攻击性内容 | 对球队/球员/国家的攻击或歧视 |
| 版权素材 | 未授权的图片、视频、文章全文 |

## Commit Message 规范

使用标准前缀，中英文间留空格：

```
feat: 新增球员对比分析模板
fix: 修正 SOURCE_POLICY 中的优先级描述
docs: 优化 README 展示结构
refactor: 重构赛前分析工作流步骤
style: 统一表格格式
```

## Pull Request 建议格式

```markdown
## 变更说明

[简要描述你做了什么]

## 变更类型

- [ ] 新增模板/示例
- [ ] 修正错误
- [ ] 文档改进
- [ ] 新角色模式
- [ ] 其他

## 检查清单

- [ ] 示例文件命名符合规范（mock_/real_）
- [ ] 虚构示例包含免责声明
- [ ] 真实案例标注数据来源
- [ ] 不包含博彩相关内容
- [ ] 概率使用区间表达
- [ ] 符合 SOURCE_POLICY.md 规范
```
