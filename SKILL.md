# 世界杯智能分析 Skill

## 概述

本 Skill 提供世界杯全周期智能分析能力，核心链路：

**球队画像 → 赛前情报 → 战术对位 → 关键球员 → 比赛预测 → 爆冷预警 → 赛后复盘 → 内容生成**

**零 API 依赖**：本 Skill 不要求用户额外配置任何外部 API。涉及实时信息时，默认联网搜索权威来源进行核验。

## 核心功能模块

### 1. 赛前情报
- 双方近期战绩、伤病名单、阵型变化
- 历史交锋记录与胜率统计
- 主教练战术偏好与临场调整风格

### 2. 球队画像
- 球队风格标签（控球型 / 反击型 / 高压逼抢型等）
- 核心球员依赖度分析
- 小组赛与淘汰赛表现差异

### 3. 球员影响力分析
- 关键球员攻防贡献量化
- 状态曲线（近5场数据趋势）
- 对位克制关系

### 4. 战术解读
- 阵型拆解与变阵预判
- 进攻与防守体系分析
- 定位球套路识别

### 5. 战术对位矩阵
- 双方阵型匹配度评估
- 关键对位点（边路 / 中场 / 锋线）
- 潜在弱侧与突破口

### 6. 可解释预测
- 胜平负概率分布（基于多维度加权）
- 比分区间预测
- 进球分布时段预测
- 预测依据与不确定因素说明

### 7. 爆冷预警
- 历史爆冷模式匹配
- 当前赛事冷门触发因子识别
- 风险等级评估（低 / 中 / 高）

### 8. 赛后复盘
- 关键事件时间线
- 战术执行度评分
- 转折点分析与假设推演

### 9. 小组出线推演
- 积分、净胜球、交锋记录多情景模拟
- 各队出线概率动态计算
- 末轮博弈分析

### 10. 内容生成
- 赛前预告文案（公众号 / 短视频 / 社交媒体）
- 赛后复盘长文
- 短视频解说脚本
- 数据可视化文案配合

## 数据来源与核验策略

本 Skill 默认不要求用户配置外部 API，实时信息按以下优先级获取：

1. 用户提供的数据
2. FIFA 官网、各大洲足联官方数据
3. 球队或足协官方公告与社交媒体
4. 权威体育媒体（ESPN、BBC Sport、Goal、懂球帝、直播吧）
5. 数据平台（FBref、Transfermarkt、WhoScored）
6. 历史赛事资料
7. 模型已有知识（仅作为补充，不作为实时数据来源）

## 约束规则

1. **不虚构实时数据**：所有统计必须标注来源或注明"截至XX日"
2. **联网核验**：涉及最新比分、积分榜、首发、伤停等必须联网确认，不凭记忆编造
3. **可解释预测**：预测结果以概率区间表达，必须说明依据和不确定性，不做绝对化断言
4. **禁止博彩建议**：不提供任何赌博、投注金额或下注建议
5. **官方优先**：伤病等敏感信息以官方通报为准，信息冲突时优先采用官方来源

## 使用方式

根据需求加载对应 prompt 模式：

| 模式 | 适用场景 |
|------|----------|
| `分析师模式` | 深度数据驱动，适合专业分析 |
| `教练模式` | 战术视角，侧重阵型与对位 |
| `解说员模式` | 生动叙事，适合内容创作 |
| `数据模式` | 纯数据输出，表格优先 |
| `媒体创作模式` | 面向公众号、短视频、社交平台 |
